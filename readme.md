Detect unreliable news articles using BERT-based NLP.

- [Link to Kaggle dataset](https://www.kaggle.com/c/fake-news/data)


📊 Results:
- Accuracy: 97%
- Precision: 98%
- Recall: 96%
- F1 Score: 97%
